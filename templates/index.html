Supply Chain Security

SBOM Risk Analyzer

Software Supply Chain Risk Scorer using Graph-Based Dependency Analysis

This application analyzes your software dependencies end-to-end — mapping transitive dependency graphs, cross-referencing known vulnerabilities, flagging outdated or unmaintained packages, and detecting license compatibility conflicts — to produce a unified risk score per application.

⚠

Vulnerability Detection

Cross-references every dependency against known CVEs, including transitive ones.

⚖

License Compatibility

Flags GPL, AGPL, and unknown licenses that introduce legal/compliance risk.

☉

Dependency Graph Analysis

Builds a directed graph of app-to-library relationships using NetworkX.

📊

Risk Score Calculation

Combines CVSS, license, maintenance, and depth penalties into one score.

Developed using Flask, NetworkX, Pandas and Plotly
